Guard Patrolling System

October 20, 2023
Scroll right-left for more previews, click to enlarge

This is my final year project; which involved transforming a traditional guard patrolling system into a comprehensive cloud-based solution. Leveraging an array of AWS services, I developed a robust and scalable architecture:

  • Networking & Security: Established a secure environment using Amazon VPC and Cloudflare.
  • IoT: Utilized AWS IoT Core to manage and communicate with ESP32 devices for real-time data collection.
  • Compute: Deployed Amazon EC2 instances for application logic and Linux environments.
  • Databases: Employed both Amazon RDS (MariaDB) for structured data and Amazon DynamoDB for flexible NoSQL storage.
  • Serverless: Integrated AWS Lambda for event-driven functions.

Additional Technologies:

  • Git: For effective version control.
  • Python & PHP: For scripting and back-end development.

This project showcases my expertise in cloud architecture design, IoT integration, and a variety of programming tools.
